# Edge-AI-Security
This paper proposes Multi-Layer Telemetry Correlation (MLC), a passive runtime integrity verification framework for Edge AI systems. The framework combines CPU, GPU/NPU, thermal, memory, scheduler, power, and inference-latency telemetry into a unified trust score for anomaly detection. The work evaluates MLC against multiple simulated attack classes including cryptomining, hidden inference, beacon malware, and firmware-level anomalies. The study is presented as a simulation feasibility analysis with a roadmap toward hardware-grounded validation on Jetson and Raspberry Pi platforms.
